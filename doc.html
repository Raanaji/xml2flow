Get Started

First of all, why not make a simple post on how to run the file? This because it is not so easy. After installing xmlstarlet, navigate to your folder containing the XML file and run the following command. Experiment with the file vanilla_swap.xml included in the repo and see the magic!

Execution

              
                xmlstarlet el -u file_name.xml|awk -F'/' 'BEGIN{print "digraph{"}{print $(NF-1)" -> "$NF}END{print"}"}'|dot -Tpng -o file_name.png

Configuration

Learn how to configure settings for xml2flow.

Dependency Reason Required
graphviz Required partially for processing the .dot file. Yes

More dependencies might be needed once additional FPML functionalities will be added.

Issues

Please ensure that your xml file has a proper hierarchichal structure, only then can the progarm run and give a proper .png output.

Developments

To ensure that multiple types and shapes of flowcharts be formed, I am working on a reconfiguring an ancient .awk file which will allow users to get desired flowcharts.

xml2flow is an initiative started by Tanishq Chauhan, because he spent an entire day banging his head trying to figure out how to create a flowchart out of 834 lines of xml banking code. If you think this was useful, follow Tanishq Here and contribute to this project.
